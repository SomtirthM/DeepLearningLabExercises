$S_{w, b}(x) = \frac{1}{1 + e^{-(wx + b)}}$¶

In [2]:
def sigmoid(x, w, b):
    return 1/(1 + np.exp(-(w*x + b)))

Let us now try the sigmoid for 2-D.

$S_{w_1, w_2, b}(x_1, x_2) = \frac{1}{1 + e^{-(w_1x_1 + w_2x_2 + b)}}$¶

In [6]:
def sigmoid_2d(x1, x2, w1, w2, b):
    return 1/(1 + np.exp(-(w1*x1 + w2*x2 + b)))
